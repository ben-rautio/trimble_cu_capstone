◆ add_scheduled_event()

void add_scheduled_event ( uint32_t  event)

Adds a particular event to be scheduled.

Bitwise ORs the parameter event into the event_scheduled variable so that it can be processed.

Note
Uses CORE statements to have the process atomic; i.e. un-interruptible.
Parameters
[in]eventA binary value to be added to the event_scheduled variable.

◆ remove_scheduled_event()

void remove_scheduled_event ( uint32_t  event)

Removes a particular event from the schedule.

Bitwise ANDs the negation of event with the event_scheduled variable to remove that particular event from the schedule.

Note
Uses CORE statements to have the process atomic; i.e. un-interruptible.
Parameters
[in]eventA binary value to be removed from the event_scheduled variable.
